Blood Alloy

Lizzi is a game in which you explore a contiguous world and make friends with other creatures and solve puzzles.

It's a personal project that I have been working on over a year using Unity3D and C#. I did the programming, 3D/2D artwork, game design, lighting, and shaders(Cg) for the game, as well as a custom level editor. Music for the game was created by Svetlana Tovarisch.

Also, the titular character is a cute lizard and she's totally not a frog or anything.

Download: http://evajolli.itch.io/lizzi

Air Brawl
Lizzi Editor is a 3D/2D tile-based level editor created in Unity3D/C# that is used in most of my current game projects. The editor runs in-game so it's immediately playable. It's also a "level manager" that handles level streaming and auto-tiling.
